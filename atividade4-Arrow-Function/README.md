# Calculadora de Combustível

Projeto simples feito com HTML, CSS e JavaScript para calcular o valor total do abastecimento com base no tipo de combustível e na quantidade de litros.

## Funcionalidades

* Seleção do tipo de combustível
* Cálculo automático do valor total
* Validação de campos vazios
* Validação de números inválidos
* Formatação em moeda brasileira
* Mantém os dados na tela ao pressionar Enter

## Tecnologias Utilizadas

* HTML
* CSS
* JavaScript

## Importante

Este projeto utiliza CSS Nesting (estilo parecido com Sass/SCSS), recurso suportado apenas em navegadores mais modernos.

Por isso, é recomendado utilizar a versão mais recente de navegadores como:

* Google Chrome
* Microsoft Edge
* Mozilla Firefox
* Safari

Caso o navegador esteja desatualizado, alguns estilos podem não funcionar corretamente.

## Como Executar

1. Clone o repositório
2. Abra a pasta do projeto
3. Execute o arquivo `index.html`
